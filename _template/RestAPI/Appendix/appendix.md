### Response Body

#### Properties

| Property Name | Description | Type / Value | Optional |
| --- | --- | --- |
| myprop | Tells you something | number | No |
| myobject | Contains info on something | object | No |
| info1 | Tells you stuff | string | No |
| info2 | Tells you things | string | Yes |

#### Example

```json
{
    "myprop": 4,
    "myobject": {
        "info1": "stuff",
        "info2": "things"
    }
}
```
