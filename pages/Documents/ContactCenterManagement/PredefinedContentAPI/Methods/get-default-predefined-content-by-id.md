Retrieves a single, default Predefined Content item by template ID from a specific account.

### Request

| Method | URL |
 |:-------- | :-----|
| GET | https://[{domain}](/agent-domain-domain-api.html)/api/configuration/defaults/engagement-window/canned-responses/{templateId} |

**Path Parameters**

 |Parameter  |Description  |Type / Value |
| :-------- | :---------- | :----------- |
 |templateId | ID of a template used to instantiate the object | string |

### Response

**Response Codes**

| Code  |Description |
 |:-----  |:------------- |
| 200|  OK|
| 404 | Data Not Found |
