### Description

Get a list of all of the Categories on an account.

### URI

/api/account/{accountId}/configuration/le-categories/categories?v=2.0

### HTTP Methods

GET

### Response Codes

200 OK

400 Bad Request

401 Not Authorized

404 Data not found

500 Internal server error

### Formats

JSON
