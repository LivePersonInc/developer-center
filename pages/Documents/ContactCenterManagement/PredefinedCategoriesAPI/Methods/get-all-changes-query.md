### Description

Query changes in Categories data for a range of revisions.

### URI

/api/account/configuration/le-categories/categories/query?v=2.0

### HTTP Methods

POST

### Response Codes

200 OK

400 Bad Request

401 Not Authorized

500 Internal server error
