In this page you can refer to some examples of the events notifications received from the Webhooks service in regards to the conversation.

### Consumer created an new conversation

```json
{
 "kind": "notification",
 "body": {
  "changes": [
   {
    "type": "UPSERT",
    "result": {
     "convId": "5491244f-e462-4c63-bf4e-faf4b2d3e7b5",
     "conversationDetails": {
      "participants": [
       {
        "id": "ade26082b1185839d17e0794924c1a0005cf5663e7c35e38f80dd8552a1a96e5",
        "role": "CONSUMER"
       }
      ],
      "brandId": "64467156",
      "state": "OPEN",
      "startTs": 1528371161882,
      "metaDataLastUpdateTs": 1528371161882,
      "ttr": {
       "ttrType": "PRIORITIZED",
       "value": 600
      }
     }
    }
   }
  ]
 },
 "type": "cqm.ExConversationChangeNotification"
}
```

     "id": "ade26082b1185839d17e0794924c1a0005cf5663e7c35e38f80dd8552a1a96e5",

### Agent is typing

```json
{
 "kind": "notification",
 "body": {
  "changes": [
   {
    "originatorId": "29c3019d-fe0b-5200-afe8-737abfeae4b4",
    "originatorMetadata": {
     "id": "29c3019d-fe0b-5200-afe8-737abfeae4b4",
     "role": "ASSIGNED_AGENT"
    },
    "event": {
     "type": "ChatStateEvent",
     "chatState": "COMPOSING"
    },
    "dialogId": "5491244f-e462-4c63-bf4e-faf4b2d3e7b5"
   }
  ]
 },
 "type": "ms.MessagingEventNotification"
}
```

**Note:** In order to also get "Consumer is typing" events notifications, the Connector must send a message using the SEND endpoint, which contains the `COMPOSING` ChatStateEvent. See more information in [Send Chat State Events](connector-api-send-api-examples.html#send-chat-state-events).
